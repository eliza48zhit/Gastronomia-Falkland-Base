🇫🇰 Gastronomia de Islas Malvinas - Smart Contract (Base Mainnet)
Este es el vigesimoprimer contrato inteligente de mi coleccion profesional "Sabores de America". Las Islas Malvinas poseen una cultura culinaria marcada por el aislamiento geografico y la fuerte herencia britanica, donde el cordero (mutton) y los productos del mar son los protagonistas. Este contrato introduce el campo de Tipo de Lana, vinculando el plato con la tradicion ganadera que define la vida en el archipielago.

🛠️ Detalles Tecnicos: Sabores del Atlantico Sur On-Chain

Este despliegue aplica estandares de desarrollo robustos en Solidity:

Hito de la Serie: Vigesimoprimer bloque completado. Se utiliza la variable tipoLana para documentar el origen ovino (ej. Corriedale, Merino), un pilar de la identidad local.

Integridad del Codigo: Aunque el compilador 0.8.20 genera avisos informativos sobre "bugs" teoricos (como LostStorageArrayWriteOnSlotOverflow), la arquitectura lineal y simple de este contrato garantiza que no sea susceptible a dichas vulnerabilidades.

Cero Caracteres Especiales: Codigo limpio de enies y acentos para prevenir errores de visualizacion y asegurar la compatibilidad universal.

Licencia Abierta: Implementado bajo SPDX-MIT para fomentar la transparencia y el aprendizaje en el ecosistema Base.

📖 Modo de Uso

Consultar el Sur Profundo: En la pestaña Read Contract, usa consultarPlato (ID 1 para el Mutton Pie) para ver la referencia de origen registrada.

Registrar Tradicion: En la pestaña Write Contract, usa registrarPlato para añadir platos como "Fish and Chips" (Origen: Local Catch) o "Upland Goose".

Votacion Inmutable: Sistema de Likes y Dislikes para medir la popularidad de la gastronomia isleña directamente en la blockchain.
